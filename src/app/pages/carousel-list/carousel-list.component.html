ng-virtual-list supports element transformations.

The following presets are available:
- ItemTransformations.LINEAR
- ItemTransformations.SWIPE_3D
- ItemTransformations.EVENT_HORIZON
- ItemTransformations.DECK_OF_CARDS
- ItemTransformations.DECK_OF_CARDS_3D.

You can also create custom transformation sets; see implementation example.

Each default transformation preset has flexible settings, allowing you to achieve various element distribution effects.
